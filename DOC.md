# Basico Cypress
## instalação e configuração cypress

### Requisitos

- GIT bash (instalados)
- Node.JS
- npm
- Google chrome
- Visual studio Code

### Instalar

* $ `npm install` OR `npm i`

## Executar os testes 

### Desktop
* $ npm run open
* $ npm run test

### Mobile

* $ npm run open:mobile
* $ npm run test:mobile
